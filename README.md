# 📖 About

A modern Tic Tac Toe game with a futuristic glassmorphism interface.

Play against your friends or challenge multiple AI difficulty levels ranging from Easy to Impossible. Customize board sizes, track your statistics, replay previous matches, and enjoy a smooth responsive experience directly in your browser.

---

# ✨ Features

- ❌ Player vs Player
- 🤖 AI Difficulty (Easy, Medium, Hard, Impossible)
- 📏 Multiple Board Sizes (3×3, 5×5, 7×7)
- 🌙 Dark & Light Theme
- 📊 Match Statistics
- 🎥 Replay Match
- ⏱️ Turn Timer
- ⚙️ Game Settings
- 💾 Local Save
- 📱 Responsive Design
